# Todo App

A full-stack Todo application built with **Node.js**, **Express**, **TypeScript**, **Handlebars**, and **MySQL**, featuring session-based authentication and server-side rendering.

---

## How to Run This Project

### Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop) installed
- Node.js and npm installed

### Steps

1. **Start Docker**  
   Make sure the Docker Desktop application is running.

2. **Launch the containers**  
   In the first terminal, run:

   ```bash
   docker-compose up
   ```

3. **Start the development server**  
    In second terminal, run:

    ```bash
    npm run dev
    ```

4. **Open Prisma Studio**  
    In third terminal, run:
    
    ```bash
    npx prisma studio
    ```
